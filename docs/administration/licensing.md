Savvina AI Community Edition is distributed under the **Business Source License 1.1** (BSL 1.1). No license key, registration, or external server call is required — the application runs fully self-contained.

## What the BSL 1.1 allows

- **Free use** for development, testing, evaluation, personal projects, and non-commercial internal use.
- **Modification and redistribution** under the same license terms.
- On **2030-06-01** (the Change Date) the license converts automatically to **Apache License 2.0**, at which point all restrictions are removed.

---

## When you need a commercial license

A commercial license is required if you:

- Deploy Savvina AI in a **production environment for business purposes**.
- Offer Savvina AI as a **hosted or managed service** to third parties.
- Embed it into a **product or SaaS platform** you sell or charge for.

See [COMMERCIAL.md](../../COMMERCIAL.md) or contact info@savvina.ai.

---

## No license key, no licensing server

The Community Edition has no tier restrictions, no license key validation, and does not phone home. All features in the codebase are available to every user:

- Unlimited connections
- Single self-hosted user account
- All datasource adapters that ship with this release
- All LLM provider adapters

---

## First-boot setup

On a fresh deployment the first person to open the browser sees the **Create Admin Account** screen. Enter an email and a password (minimum 12 characters) to create the single user account. This screen is shown only once — the registration endpoint is closed as soon as any account exists.
